## 🌟 Overview
**AdZee.pk** is a masterfully crafted, pixel-perfect frontend template designed for massive scale **Classifieds** and **E-Commerce** markets (similar to OLX, eBay, or Craigslist). Built entirely with pure HTML5, CSS3 (Flexbox/Grid), and Vanilla JavaScript, this template delivers exceptional **UI/UX**, blazing-fast performance, and a 100% mobile-responsive design without relying on heavy frameworks like Bootstrap or Tailwind.

## ✨ Key Features
- **🚀 Ultra-Fast & Lightweight:** Pure HTML, CSS, and JS. Zero bloat.
- **📱 100% Mobile Responsive:** "Mobile-First" approach ensuring flawless rendering on all screens (iOS, Android, Tablets, Desktops).
- **🗺️ Advanced Mega Menu:** A massive, beautifully structured hover-triggered CSS Mega Menu containing dozens of categories (Cars, Real Estate, Electronics, etc.).
- **🧩 Sticky Smart Navbar:** A centered Flexbox-driven navigation bar that stays pinned to the top.
- **🔐 Dedicated Auth Pages:** Separated and beautifully designed `adzee_login.html` with a modern 2-column layout and social login UI.
- **🎯 Dynamic Hero Search:** Advanced multi-field hero search overlay exactly like premium marketplace platforms.
- **🚥 Horizontal Scroll Categories:** Smooth horizontal scrolling category slider using pure CSS.
- **💬 WhatsApp Integration:** Floating WhatsApp action button and native WhatsApp ad contact buttons.
- **🎨 Easily Customizable:** All primary brand colors and dimensions are managed via simple CSS `:root` variables.

## 🛠️ Developer Guide (How to Update)
- **🎨 Changing Colors:** Open `adzee.pk.html` or `adzee_login.html` and modify the CSS `:root` variables at the top.
  ```css
  :root {
      --red: #D32F2F;        /* Main Brand Color */
      --orange: #f0892d;     /* Action Buttons */
      --gray-light: #f5f5f5; /* Backgrounds */
  }
  ```
- **📂 Adding Categories:** You can extend the `.mega-menu` or `.cats-slider` by duplicating existing `<div class="mega-col">` or `<a class="cat-item">` tags.
- **🔌 Backend Integration (Future Scope):** This is currently a Frontend UI Template. You can easily integrate it with **Node.js, Express, React, PHP (Laravel, CodeIgniter), Django, or Python** by binding the forms and anchors to your REST API endpoints.
